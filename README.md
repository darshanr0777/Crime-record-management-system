🌟 Project Overview

The Crime Record Management System (CRMS) is a full-stack web application developed to manage crime-related records digitally and efficiently.

The system provides a centralized platform for managing:

Police Stations
Police Officers
Crime Reports
Criminal Records
Victim Details
Evidence Management
Case Tracking

The application helps police departments reduce manual paperwork, improve data accuracy, and manage crime information securely.

The system is developed using:

PHP
MySQL
HTML
CSS
JavaScript

⚙️ Installation Steps
Clone the Project
git clone https://github.com/project/crime-record-management-system.git
Move to Project Folder
cd crime-record-management-system
💻 Database Setup
Import MySQL Database
crime_record.sql
Configure Database

Update database details in:

config.php

Example:

$host = "localhost";
$user = "root";
$password = "";
$database = "crime_record";
▶️ Run the Project

Place project folder inside:

htdocs

Start:

Apache Server
MySQL Server

Open browser:

http://localhost/crime-record-management-system
