🎯 STEP-BY-STEP SETUP (5 MINUTES)

┌─────────────────────────────────────────────────────────────┐
│ STEP 1: START XAMPP                                         │
└─────────────────────────────────────────────────────────────┘
1. Open XAMPP Control Panel
2. Click "Start" for Apache
3. Click "Start" for MySQL
4. Wait for both to turn green ✅

┌─────────────────────────────────────────────────────────────┐
│ STEP 2: CREATE DATABASE                                     │
└─────────────────────────────────────────────────────────────┘
1. Open browser: http://localhost/phpmyadmin
2. Click "New" on left sidebar
3. Database name: crime_management
4. Click "Create"
5. Import your existing database tables OR create them manually

┌─────────────────────────────────────────────────────────────┐
│ STEP 3: RUN DATABASE UPDATES (IMPORTANT!)                   │
└─────────────────────────────────────────────────────────────┘
1. In phpMyAdmin, select "crime_management" database
2. Click "SQL" tab at the top
3. Open file: database_updates.sql
4. Copy ALL content and paste into SQL box
5. Click "Go" button
6. You should see success messages ✅

┌─────────────────────────────────────────────────────────────┐
│ STEP 4: CREATE ADMIN USER                                   │
└─────────────────────────────────────────────────────────────┘
1. In phpMyAdmin, click "officers" table
2. Click "Insert" tab
3. Fill in these values:
   
   user_id:     OFF001
   name:        Admin User
   role:        Administrator
   username:    admin
   contact_no:  1234567890
   email:       admin@police.com
   password:    admin123
   status:      active

4. Click "Go" to insert
5. Verify row was created ✅

┌─────────────────────────────────────────────────────────────┐
│ STEP 5: LAUNCH THE SYSTEM                                   │
└─────────────────────────────────────────────────────────────┘
1. Open browser
2. Go to: http://localhost/crime_management_system/
3. You should see the modern login page 🎨
4. Login with:
   Username: admin
   Password: admin123
5. Click "Login" button
6. You should reach the Dashboard ✅

┌─────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING                                             │
└─────────────────────────────────────────────────────────────┘

❌ PROBLEM: Login page shows but can't login
✅ SOLUTION: Make sure you created the admin user in Step 4

❌ PROBLEM: "Database connection failed"
✅ SOLUTION: 
   - Check MySQL is running in XAMPP
   - Verify database name is "crime_management"
   - Check backend/db_connect.php settings

❌ PROBLEM: Tables are empty in iframes
✅ SOLUTION: 
   - Database updates not run (see Step 3)
   - No data in tables yet - add some records

❌ PROBLEM: Blank white page
✅ SOLUTION:
   - Check Apache is running
   - Look at Apache error logs
   - Enable PHP error display

❌ PROBLEM: Can't add records
✅ SOLUTION:
   - Verify all database tables exist
   - Check status column exists (run database_updates.sql)
